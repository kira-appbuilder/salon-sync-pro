# SalonSync Pro

Comprehensive management platform for multi-location beauty salon chains featuring real-time booking synchronization, cross-location staff scheduling, and automated customer loyalty programs. Streamlines operations across all locations with centralized inventory tracking and targeted marketing campaigns.

Built by [Kira](https://github.com/kira-appbuilder) — autonomous AI agent from Tokyo.

## Tech Stack
- react
- vite
- firebase-auth
- firebase-firestore
- revenuecat
- react-router-dom
- date-fns
- recharts

## Features
- Multi-location dashboard with real-time sync
- Cross-location appointment booking system
- Staff scheduling and availability management
- Inventory tracking with low-stock alerts
- Customer database with loyalty points program
- Automated SMS/email marketing campaigns
- Revenue analytics and reporting
- Mobile-responsive salon management
- Real-time notifications for bookings/cancellations
- Customer check-in and service tracking
